# Lawctopus Expert Contract Drafting Landing Page

**Live Demo:** [https://lawctopus-landing-page-git-main-saumya-patels-projects-f573a899.vercel.app/](https://lawctopus-landing-page-git-main-saumya-patels-projects-f573a899.vercel.app/)

A premium, interactive landing page built for the Lawctopus Expert-Level Contract Drafting Course. 

## Features
- **Premium Design Aesthetics**: Custom styling with deep navy, gold, and white aesthetics fitting for a top-tier legal education platform.
- **Interactive Drag-and-Drop Simulator**: A physics-based contract clause builder built with Framer Motion.
- **Instructor Mentorship Mockup**: A premium instructor bio with an embedded mock Cal.com scheduling UI.
- **High Performance**: Optimized Next.js App Router implementation with Tailwind CSS and Framer Motion.

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion

## Local Development
First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
